Слученкова Мария ИДБ-16-07

Идентификатор прецедента: Лабораторная работа №1/ВКР
Название прецедента: Деятельность технической поддержки программных продуктов
Контекст: Работа технической поддержки

Участники и цели:

  • участник - 1-я линия, категория - основной, цель - обработка заявки и выполнение по возможности;
  • участник - 2-я линия, категория - основной, цель - успешное выполнение заявки;
  • участник - 3-я линия, категория - внешний, цель - содействие 2-ой линии в выполнении заявок;
  • участник - аналитик, категория - основной, цель - проанализировать данные по заявкам и передать данные руководителю;
  • участник - руководитель, категория - внешний, цель - увеличить эффективность работы технической поддержки;
  • участник - ServiceDesk, категория - инструмент, цель - предоставить сервис для работы в нем;
  • участник - БД/СУБД, категория - инструмент, цель - предоставить средства для анализа заявок.

Предусловия: наличие любой заявки в системе

Постусловия: рабочий день окончен

Основной поток
